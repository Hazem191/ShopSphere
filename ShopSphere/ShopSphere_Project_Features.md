# ShopSphere - متجر إلكتروني متكامل (ASP.NET Core MVC)

متجر **ShopSphere** هو منصة تجارة إلكترونية احترافية تم تطويرها باستخدام تقنية **ASP.NET Core MVC**. يتميز المشروع بتصميم "بريميوم" عصري، ودعم كامل للغات، ونظام إدارة متكامل للمستخدمين والمسؤولين.

---

## 🚀 التقنيات المستخدمة (Tech Stack)
*   **Backend:** ASP.NET Core 8.0 MVC.
*   **Database:** Entity Framework Core (SQL Server).
*   **Security:** Microsoft Identity لغدارة المستخدمين والأدوار.
*   **Frontend:** HTML5, CSS3 (Vanilla), JavaScript, Bootstrap 5.
*   **Localization:** Microsoft Localization (دعم لغات متعددة).
*   **Persistence:** Session Management لتخزين مسودة السلة.

---

## 🌟 مميزات النظام (Core Features)

### 1. تجربة المستخدم (UX/UI)
*   **تصميم عصري وجذاب:** استخدام ألوان متناسقة، وتأثيرات زجاجية (Glassmorphism)، وانتقالات سلسة.
*   **الوضع الداكن والفاتح (Dark/Light Mode):** دعم كامل لتغيير السمة مع حفظ الاختيار في المتصفح.
*   **تصميم متجاوب:** يعمل المتجر بكفاءة على جميع الأجهزة (موبايل، تابلت، ديسكتوب).
*   **نظام التنبيهات (Toast Notifications):** استخدام توستات احترافية لإبلاغ المستخدم عند إضافة منتج أو تنفيذ طلب.

### 2. كتالوج المنتجات (Product Catalog)
*   **تصفح المنتجات:** عرض المنتجات في كروت أنيقة تحتوي على كافة التفاصيل الأساسية.
*   **البحث المتقدم:** إمكانية البحث عن المنتجات بالاسم أو الوصف.
*   **التصنيف (Categories):** تصفية المنتجات بناءً على الفئات.
*   **الفرز (Sorting):** ترتيب المنتجات حسب الأحدث، السعر (من الأقل للأعلى أو العكس).
*   **صفحة التفاصيل:** عرض كامل لوصف المنتج، المخزن، والمراجعات.

### 3. نظام التسوق (Shopping System)
*   **سلة التسوق (Cart):** إضافة المنتجات، تعديل الكميات، وحذف المنتجات مع تحديث فوري للمخزن في الواجهة.
*   **قائمة الأمنيات (Wishlist):** نظام قلب تفاعلي يسمح للمستخدم بحفظ منتجاته المفضلة للرجوع إليها لاحقاً.
*   **نظام السداد (Checkout):** عملية شراء بسيطة تطلب من المستخدم إدخال عنوان الشحن وتأكيد الطلب.
*   **تتبع الطلبات (Order Tracking):** صفحة خاصة للمستخدم لعرض تفاصيل طلباته السابقة وحالاتها (Pending, Shipped, etc.).

### 4. التفاعل والمراجعات (Reviews & Ratings)
*   **نظام التقييم:** يمكن للمستخدمين تقييم المنتجات بالنجوم وترك تعليقاتهم.
*   **عرض المراجعات:** تظهر المراجعات في صفحة المنتج مع حساب متوسط التقييم.

---

## 🌍 دعم اللغات (RTL & Localization)
*   **متعدد اللغات:** دعم كامل للغتين العربية والإنجليزية.
*   **RTL Support:** تحويل كامل لاتجهات الصفحة (Right-to-Left) عند اختيار اللغة العربية لضمان تجربة مستخدم طبيعية.
*   **إدارة الموارد:** استخدام ملفات `.resx` لسهولة تعديل النصوص والترجمات.

---

## 🔐 نظام الهوية وإدارة المستخدمين (Identity)
*   **التسجيل والدخول:** نظام آمن لإدارة حسابات المستخدمين باستخدام ASP.NET Identity.
*   **إدارة الملف الشخصي:** تعديل البيانات، تغيير كلمة المرور، وإدارة تفاصيل الحساب.
*   **تأكيد البريد الإلكتروني:** المشروع مهيأ لإرسال رسائل البريد الإلكتروني لتأكيد الحسابات أو استعادة كلمة المرور عبر `EmailSender`.

---

## 🛠️ لوحة تحكم الإدارة (Admin Panel) - للمسؤولين فقط
*   **إدارة المنتجات:** (إضافة، تعديل، حذف، عرض) مع رفع الصور وضبط SKU والأسعار.
*   **إدارة الفئات:** التحكم في تصنيفات المتجر.
*   **إدارة الطلبات:** عرض جميع طلبات العملاء وتغيير حالاتها لمتابعة عملية الشحن.

---

## 🎨 لمسات احترافية إضافية
*   **تحديث المخزن الفوري:** الواجهة تتفاعل مع السلة وتظهر "المتبقي" في المخزن ديناميكياً.
*   **تحسين Breadcrumbs:** مسار تنقل ذكي يتكيف مع اتجاه اللغة.
*   **إصلاحات CSS مخصصة:** ضمان عدم تداخل النصوص في حقول الإدخال، خاصة في الأجهزة الصغيرة.
